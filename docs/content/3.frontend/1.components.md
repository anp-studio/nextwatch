## Pages

Vue pages in `app/pages/`:

### index.vue

Main landing/home page. Shows:

- Featured movies
- User recommendations
- Quick actions

### login.vue

Authentication page. Handles:

- Email/password login
- Sign up form
- Forgot password link

### profile.vue

User profile page. Displays:

- User information
- Watch statistics
- Personalization settings

### search.vue

::note
Search method needs improvement
::

Movie search and browse. Features:

- Search by title
- Filter by genre
- Display results with pagination

### watched.vue

User's watch history. Shows:

- Movies user has watched
- Ratings given
- Option to remove from history

### reset-password.vue

Password recovery flow:

- Email verification
- New password creation
- Confirmation

## Components

Reusable Vue components in `app/components/`:

### MovieDetails.vue

Displays detailed movie information:

- Title, poster, synopsis
- Cast and crew
- Rating and metadata
- User actions (rate, watch, add to list)

## Composables

Shared logic in `app/composables/`:

### useAuth.ts

Authentication management:

```typescript
const { user, login, logout, signup } = useAuth()
```

### useMovies.ts

Movie data fetching:

```typescript
const { movies, search, getDetails } = useMovies()
```

### useSupabase.ts

Supabase integration:

```typescript
const { supabase, user, session } = useSupabase()
```
