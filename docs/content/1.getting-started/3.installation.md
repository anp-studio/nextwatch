## Prerequisites

Make sure you have the following installed:

- **Node.js** (v18 or higher)
- **Python** (v3.8 or higher)
- **Git**
- **npm**

## Installation Steps

::steps
### Clone the Repository

```bash [Terminal]
git clone https://github.com/movie-recommender-yh4f/movie-recommender.git
cd movie-recommender
```

### Setup Frontend

Navigate to the frontend directory and install dependencies:

```bash [Terminal]
cd frontend
npm install
```

Configure environment variables by creating a `.env.local` file:

```bash [Frontend .env.local]
NUXT_PUBLIC_SUPABASE_URL=your_supabase_url
NUXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_key
```

### Setup Backend/Model

Navigate to the model directory:

```bash [Terminal]
cd ../model
pip install -r requirements.txt
```

### Start Development Server

Back in the frontend directory, start the development server:

```bash [Terminal]
cd ../frontend
npm run dev
```

The application will be available at `http://localhost:3000`
::

## Verify Installation

Once everything is set up, you should be able to:

1. Open `http://localhost:3000` in your browser
2. See the Movie Recommender interface
3. Click on movies and browse recommendations

## Troubleshooting

If you encounter any issues:

- Ensure all dependencies are correctly installed
- Check that environment variables are properly configured
- Verify that Node.js and Python versions meet the requirements
- Check the [Troubleshooting](/getting-started/troubleshooting) guide for common issues
