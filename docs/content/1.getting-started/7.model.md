## Components

Movie Recommender backend has three parts:

1. **ML Model** (Python) - Recommendation engine
2. **API Server** - REST API
3. **Database** (Supabase) - User & rating data **(WIP)**

## Quick Setup

### Model

```bash
cd model
pip install -r requirements.txt
python scripts/process_data.py
```
