## Data Pipeline

The model processes raw TMDB data through several stages:

### Stage 1: Raw Data

TMDB exports are loaded from:

- `data/raw/TMDB_all_movies.csv` - Complete movie dataset
- `data/raw/TMDB_tv_dataset_v3.csv` - TV shows dataset

### Stage 2: Cleaning

Data cleaning steps:

- Remove duplicates
- Handle missing values
- Standardize formats
- Validate data types

### Stage 3: Processing

Processed data outputs:

- `movies_core.csv` - Main movie reference
- `movies_cold.csv` - Cold-start movies for new users
- `shows_normalized.csv` - Normalized TV shows

### Stage 4: Feature Extraction

Extract features from movies:

- Genres (one-hot encoded)
- Release year
- Director and cast information
- TMDB ratings
- Runtime

### Stage 5: Similarity Calculation

Compute similarity matrices between movies:

- Cosine similarity based on features
- Stores in `similarity_matrix_optimized.csv`
- Pre-computed for fast lookups

## Files Used

All data processing scripts are in `model/scripts/`:

```text
model/
├── data/
│   ├── raw/              # Original TMDB CSVs
│   └── processed/        # Cleaned and computed data
├── scripts/
│   └── 1_run_all_normalization.py   # Main processing script
```

## Processing Script

Run data processing:

```bash [Terminal]
cd model
python scripts/1_run_all_normalization.py
```

This generates all processed data files needed for recommendations.
