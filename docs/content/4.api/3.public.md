# Public API

Public endpoint that doesn't require authentication. This is rate-limited to prevent abuse.

## Rate Limiting

**Limit:** 100 requests per minute per IP address

When rate limited, the response includes:

```text
HTTP 429 Too Many Requests
X-RateLimit-Limit: 100
X-RateLimit-Remaining: 0
```

## Get Similarities

Get movies and TV shows similar to a given movie.

### Endpoint

```text
GET /api/public/similarities/:movieId
```

### Path Parameters

- **movieId** (required) - The TMDB movie ID

### Query Parameters

- **limit** (optional) - Maximum number of results (default: all)
- **minScore** (optional) - Minimum similarity score (0-1)

### Example Requests

**Basic request:**

```bash
curl https://api.example.com/api/public/similarities/550
```

**With filters:**

```bash
curl "https://api.example.com/api/public/similarities/550?limit=10&minScore=0.8"
```

**Using JavaScript:**

```js
// Fetch similar movies for Fight Club (ID: 550)
const response = await fetch('/api/public/similarities/550')
const data = await response.json()

console.log(data)
// Output:
// {
//   movieId: 550,
//   count: 25,
//   similarities: [
//     { id: 278, type: 'movie', score: 0.82877874 },
//     { id: 1291, type: 'movie', score: 0.82542 },
//     ...
//   ]
// }
```

**With options:**

```js
// Get top 5 similar movies with score >= 0.8
const params = new URLSearchParams({
  limit: '5',
  minScore: '0.8'
})
const response = await fetch(`/api/public/similarities/550?${params}`)
const data = await response.json()
```

### Response Format

```json
{
  "movieId": 550,
  "count": 5,
  "similarities": [
    {
      "id": 278,
      "type": "movie",
      "score": 0.952
    },
    {
      "id": 1291,
      "type": "show",
      "score": 0.824
    }
  ]
}
```

### Response Fields

- **movieId** - The input movie ID
- **count** - Number of results returned
- **similarities**- Array of similar items
  - **id** - TMDB ID of similar movie/show
  - **type** - Either `"movie"` or `"show"`
  - **score** - Similarity score (0-100)

### Error Responses

**Invalid movie ID:**

```json
{
  "statusCode": 400,
  "message": "Invalid movie ID format"
}
```

**Movie not found in similarity data:**

```json
{
  "statusCode": 404,
  "message": "No similarities found for this movie"
}
```

**Rate limit exceeded:**

```json
{
  "statusCode": 429,
  "message": "Too many requests"
}
```

## Performance

Results are cached for 24 hours to ensure fast responses. Data is loaded into memory on server startup.
