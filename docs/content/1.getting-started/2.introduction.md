Welcome to **Movie Recommender**, a web app for browsing and getting movie recommendations.

::callout{type="info"}
Use this section to get oriented quickly, then jump into API and database pages for implementation details.
::

## What is This?

Movie Recommender is organized into three main directories:

- **App** (`app/`) - Nuxt 4 full-stack app with Vue and TailwindCSS
- **Database** (`database/`) - SQLite database with movie data
- **Docs** (`docs/`) - Developer documentation (what you're reading now)

The app contains both client and server code: pages, components, API routes, auth, and recommendation logic.

## Documentation Structure

This documentation covers:

- **Getting Started** - Setup and project layout
- **Backend API** - Server routes and how they work
- **Database** - Supabase and SQLite storage
- **Frontend** - Vue components and pages
