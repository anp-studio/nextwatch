# User API

Protected endpoints that require Supabase authentication.

## Authentication

All user API endpoints require a valid Supabase session. The user must be logged in.

### How to Authenticate

Authentication information is automatically included via cookies or headers when using the frontend composables:

```js
// In any Vue component using the useMovies composable
import { useMovies } from '#app'

export default {
  setup() {
    const { getUserSimilarities } = useMovies()
    
    // This automatically handles authentication
    const recommendations = await getUserSimilarities(movieId)
  }
}
```

### Endpoint

```text
GET /api/user/similarities/:movieId
```

### Path Parameters

- **movieId** (required) - The TMDB movie ID

### Query Parameters

- **limit** (optional) - Maximum number of results
- **minScore** (optional) - Minimum similarity score filter

### Response Format

```json
{
  "movieId": 550,
  "count": 100,
  "similarities": [
    {
      "id": 278,
      "type": "movie",
      "score": 0.912
    },
    {
      "id": 1502,
      "type": "show",
      "score": 0.875
    }
  ]
}
```

### Error Responses

**Not authenticated:**

```json
{
  "statusCode": 401,
  "message": "Unauthorized: User not authenticated"
}
```

**Invalid movie ID:**

```json
{
  "statusCode": 400,
  "message": "Invalid movie ID format"
}
```

**User not found:**

```json
{
  "statusCode": 404,
  "message": "User profile not found"
}
```

## Difference from Public API

| Feature        | Public API | User API           |
| -------------- | ---------- | ------------------ |
| Authentication | No         | Requires Supabase  |
| Rate Limit     | 100/min    | Standard           |
| User Data      | Generic    | Personalized (WIP) |
| Profile Access | No         | Yes                |

## Session Management

User sessions are managed by Supabase:

```js
import { useSupabase } from '#app'

export default {
  setup() {
    const { auth } = useSupabase()
    
    // Check if user is logged in
    const isLoggedIn = computed(() => auth.value.user !== null)
    
    return { isLoggedIn }
  }
}
```
