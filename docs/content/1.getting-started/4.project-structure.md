## Backend Server

The backend is built into the frontend Nuxt application using Nuxt server capabilities. It handles API routes, authentication, caching, and external service integration.

**Key Responsibilities:**

- **API Routes**: Expose endpoints for frontend and external clients
- **TMDB Integration**: Proxy requests to The Movie Database API
- **Recommendations**: Serve pre-computed movie/show similarities
- **Authentication**: Validate admin and user requests
- **Rate Limiting**: Protect public endpoints from abuse
- **Caching**: In-memory caching of similarity data (24-hour TTL)

See [Backend Server](/getting-started/backend) for detailed documentation.
