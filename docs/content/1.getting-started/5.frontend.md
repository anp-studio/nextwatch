## Frontend Setup

Located in `frontend/` directory.

### Install

```bash
cd frontend
npm install
```

### Environment Variables

Create `.env.local`:

```text
NUXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NUXT_PUBLIC_SUPABASE_ANON_KEY=your-key

# Supabase tokens
NUXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NUXT_PUBLIC_SUPABASE_ANON_KEY=your-key

# Aliases needed by server-side middleware and storage utils
SUPABASE_URL=https://your-project.supabase.c
SUPABASE_KEY=your-key
SUPABASE_SERVICE_KEY=your-key

# TMDB token
NUXT_TMDB_API_KEY=your-tmdb-key

# Admin API token
ADMIN_API_TOKEN=secure_token_here
```

### Run Dev Server

```bash
npm run dev
```

Available at `http://localhost:3000`

### Build

```bash
npm run build
npm run preview
```
