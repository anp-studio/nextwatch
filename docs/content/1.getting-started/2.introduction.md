Welcome to **Movie Recommender**, a machine learning-based recommendation system.

## What is This?

Movie Recommender is a project that implements recommendation algorithms using TMDB movie data. It's a full-stack system with:

- **Backend API** - Serves recommendations
- **Frontend** - Nuxt 4 with Vue and TailwindCSS
- **ML Model** - Python-based similarity calculations
- **Database** - PostgreSQL via Supabase

## Documentation Structure

This documentation focuses on **technical implementation** for developers:

- **Model & Data** - Data processing, TMDB integration, similarity matrices
- **AI & Algorithms** - Collaborative filtering, content-based filtering, hybrid approach
- **Frontend** - Vue components, pages, routing, composables
