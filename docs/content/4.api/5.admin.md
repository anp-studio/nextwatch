# Admin API

Protected administrative endpoints for uploading similarity matrix. All admin endpoints require a valid admin token.

## Authentication

All admin endpoints require the `X-Admin-Token` header with a valid token.

### Getting Started

1. Set your admin token in environment variables:

```bash
ADMIN_API_TOKEN=your_secure_random_token_here
```

2. Include the token in all admin requests:

```bash
curl -H "X-Admin-Token: your_secure_random_token_here" \
  https://api.example.com/api/admin/upload-similarities
```

3. Or using JavaScript:

```js
const headers = {
  'X-Admin-Token': 'your_secure_random_token_here',
  'Content-Type': 'application/json'
}

await fetch('/api/admin/upload-similarities', {
  method: 'POST',
  headers,
  body: JSON.stringify(data)
})
```

## Upload Similarities

Update the movie/show similarity matrix with new data.

### Endpoint

```text
POST /api/admin/upload-similarities
```

### Headers

- **X-Admin-Token** (required) - Valid admin authentication token
- **Content-Type** - `application/json`

**Columns:**

- **id** - Source movie/show TMDB ID
- **similarId** - Similar movie/show TMDB ID
- **type** - 0 for movie, 1 for TV show
- **score** - Similarity score (0-100)

### Response Format

**Success:**

```json
{
  "statusCode": 200,
  "message": "Similarities uploaded successfully",
  "count": 15432
}
```

**Invalid Token:**

```json
{
  "statusCode": 403,
  "message": "Forbidden: Invalid or missing admin token"
}
```

**Invalid Data:**

```json
{
  "statusCode": 400,
  "message": "Invalid CSV format or corrupted data"
}
```

## Cache Management

The system maintains a 24-hour cache of similarity data. After uploading new data:

- Cache is automatically invalidated
- Next request will reload the new data
- No manual cache clearing needed

## Security Best Practices

### Token Management

1. Generate secure random token:

```bash
# On Unix/Linux
openssl rand -base64 32

# On PowerShell
$bytes = New-Object byte[] 32
[System.Security.Cryptography.RNGCryptoServiceProvider]::Create().GetBytes($bytes)
[Convert]::ToBase64String($bytes)
```

2. Never commit token to version control
3. Use environment variables in production
4. Rotate tokens periodically

### Environment Variables

```bash
# .env (never commit this file)
ADMIN_API_TOKEN=your_secure_token_here
```

## Troubleshooting

### 403 Forbidden Error

```bash
# Verify token is correct
export ADMIN_API_TOKEN="your_token_here"
echo "Token: $ADMIN_API_TOKEN"

# Test with curl
curl -v -H "X-Admin-Token: ${ADMIN_API_TOKEN}" \
  https://api.example.com/api/admin/upload-similarities
```

### CSV Parse Error

- Ensure CSV has proper headers
- Check for special characters in values
- Verify all required columns present
- Use proper escaping for quoted fields

### Large File Timeout

- Split large files into smaller batches
- Increase server timeout settings
- Monitor network conditions
