## Installation

### Python Issues

```bash
# upgrade pip
python -m pip install --upgrade pip

# create venv
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows

# install deps
pip install -r requirements.txt
```

### Node Issues

```bash
# clear cache
npm cache clean --force

# reinstall
rm -rf node_modules package-lock.json
npm install
```

## Development Server

### Port 3000 in use

```bash
# use different port
npm run dev -- --port 3001

# or kill process on port 3000
lsof -ti:3000 | xargs kill -9  # Linux/Mac
```

### Supabase connection errors

- Check `.env.local` exists in `frontend/`
- Verify `NUXT_PUBLIC_SUPABASE_URL` is correct
- Verify `NUXT_PUBLIC_SUPABASE_ANON_KEY` is correct
- Check Supabase project is active

## Build Issues

### Cannot find module errors

```bash
# rebuild
rm -rf .nuxt
npm run build
```

### Slow build

- Close other apps
- Check disk space
- Run on SSD for faster I/O

## Data Processing

### Similarity matrix not computing

- Check raw data in `data/raw/` exists
- Ensure 4GB+ RAM available
- Check disk space

### Memory issues

- Process smaller subsets first
- Check available system memory
- Run on machine with more RAM

### API Endpoints Returning 404

**Problem**: API calls fail with 404 errors

**Solution:**

- Verify backend server is running
- Check API base URL in configuration
- Ensure correct endpoint paths

## Data Processing Issues

### Similarity Matrix Not Computing

**Problem**: Data processing script hangs or fails

**Solution:**

- Ensure raw data files exist in `data/raw/`
- Check available disk space
- Verify Python has access to read files
- Monitor memory usage - large datasets need 4GB+ RAM
- Check file permissions

### Movies Not Appearing in Search

**Problem**: Movie data seems empty

**Solution:**

- Run data processing script: `python scripts/process_data.py`
- Verify output files in `data/processed/`
- Check database has movies imported
- Restart backend server

## Common Error Messages

### "Cannot find module '@supabase/nuxt'"

**Solution:**

```bash
npm install @supabase/nuxt @supabase/supabase-js
```

### "ERR\_MODULE\_NOT\_FOUND"

**Solution:**

- Check node\_modules exists: `npm install`
- Verify tsconfig.json paths are correct
- Clear build cache: `rm -rf .nuxt`

### "EACCES: permission denied"

**Solution:**

- Use `sudo` (not recommended for security)
- Fix npm permissions using [npm docs](https://docs.npmjs.com/resolving-eacces-permissions-errors-when-installing-packages-globally)
- Use nvm (Node Version Manager) for better node management

## Performance Issues

### Slow Recommendation Loading

**Problem**: Recommendations take long to load

**Solution:**

- Precompute similarity matrices
- Add database indexes
- Enable API response caching
- Scale backend resources

## Database Issues

### Supabase Connection Timeout

**Problem**: Database queries timeout

**Solution:**

- Check network connection
- Increase query timeout settings
- Optimize slow queries with indexes

### Cannot Write to Database

**Problem**: Insert/update operations fail

**Solution:**

- Verify user is authenticated
- Check row-level security (RLS) policies
- Ensure user has permission for the table
- Verify data types match schema

## Getting Help

If issues persist:

1. Check documentation thoroughly
2. Review error logs in browser console (`F12`)
3. Check server logs in terminal
4. Submit detailed issue report with:
   - Error message
   - Steps to reproduce
   - Node/Python version
   - OS information
